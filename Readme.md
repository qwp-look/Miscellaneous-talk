## 关于本站  
> 这不是展示技术的项目，而是记录生活、整理思路的碎碎念空间。

我会记录：
- 生活小确幸／小烦恼（分享好吃的、遇到的囧事）  
- 各种折腾实录（安装系统、研究智能家居、尝试新菜）  
- 偶尔的奇思妙想（碎碎念、突然的小感悟）
